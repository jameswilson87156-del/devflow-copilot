# DevFlow Copilot 视觉参考稿

本目录存放 DevFlow Copilot 前端重设计阶段使用的 AI-generated UI concept images。

这些图片仅作为视觉参考，用于确定页面布局、配色、卡片风格、信息层级和产品展示方向，不代表项目真实运行截图。

真实运行截图应存放在 `docs/images/` 目录，并由本地前端页面通过浏览器截图脚本生成。
